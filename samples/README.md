# Synthetic Sample Data

This folder is reserved for synthetic sample input files.

Recommended synthetic sample files:
- synthetic_ar_ap.csv
- synthetic_general_ledger.csv
- synthetic_trial_balance.csv

Rules:
- No real client data.
- No exported firm data.
- No private bookkeeping records.
- Use fictional customers, vendors, accounts, dates, and amounts.
- Keep examples small enough for reviewers to understand quickly.
