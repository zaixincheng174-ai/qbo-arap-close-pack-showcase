# Review Check Categories

## AR/AP review

- AR/AP aging or balance review
- Customer/vendor-level balance checks
- AR/AP to GL control account comparison
- Missing customer/vendor identifiers
- Unusual balances or stale items

## GL review

- Uncategorized transactions
- Suspense accounts
- Clearing accounts
- Missing memo / payee / class / location
- Large or unusual transactions
- Account activity requiring review

## Trial Balance review

- Ending balance checks
- Account-level variance review
- Unexpected balance direction
- Balance movement requiring explanation

## Reconciliation and clearing

- Clearing account unresolved balances
- Undeposited funds issues
- AR/AP tie-out exceptions
- GL/TB balance consistency checks

## Ask Client list

Findings that require client clarification should be converted into clear questions.
