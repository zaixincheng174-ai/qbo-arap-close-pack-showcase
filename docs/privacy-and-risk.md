# Privacy and Public-release Notes

This public showcase is intentionally sanitized.

## Not included

- Real client data
- Real firm data
- Private accounting records
- Credentials
- Environment variables
- Internal audit folders
- Commercial strategy files
- Private workbooks
- Raw logs

## Data policy

All public examples should be synthetic or manually sanitized.

## Commercial boundary

The public repo explains the workflow and architecture. It does not expose all private implementation details, internal rulesets, mapping memory, resolution memory, or commercial strategy documents.

## Resume boundary

It is acceptable to describe this as a working prototype or private implementation with a public showcase. Do not claim production users, revenue, or client deployment unless that becomes true.
