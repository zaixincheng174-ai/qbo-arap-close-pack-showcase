# 5-minute Demo Script

## 0:00–0:30 — Problem

Small accounting teams often use QBO/Xero exports and Excel workbooks to prepare monthly close review. The problem is that AR/AP, GL, and TB review is usually manual, repetitive, and hard to trace.

## 0:30–1:15 — Upload

Show the upload flow for AR/AP export, General Ledger export, and Trial Balance export. Explain that the system is not trying to replace Excel. It produces an Excel-native review pack.

## 1:15–2:00 — Mapping and validation

Show source file role assignment and field mapping. Explain that clean mapping is required because findings must trace back to source rows.

## 2:00–3:15 — Review result

Show generated sections: exceptions, AR/AP review, GL/TB reconciliation, variance bridge, and Ask Client list.

## 3:15–4:15 — Traceability

Open one finding and explain how it traces back to source file, source row, account, and amount.

## 4:15–5:00 — Excel output

Download/open the Excel workbook. Show that the workbook is structured for reviewer handoff, not just dashboard viewing.
